# Fitbit Air Dashboard — Frontend

This is the **Next.js 16** frontend for the Fitbit Air Dashboard.

For full setup instructions, project overview, and documentation, see the **[root README](../README.md)**.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). The backend (`google-health-service/`) must also be running on port 8000 for live data.
